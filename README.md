# FitCoach : Plateforme de Réservation de Séances Sportives Personnalisées

## Contexte du Projet

Le projet **FitCoach** est une plateforme web dédiée à la mise en relation entre **sportifs** et **coachs professionnels** (football, tennis, natation, athlétisme, sports de combat, préparation physique, etc.).  

- Les **sportifs** peuvent consulter les profils des coachs, découvrir leurs spécialités et réserver des séances sportives personnalisées en ligne.  
- Les **coachs** gèrent leurs disponibilités, leurs réservations et leur profil professionnel.


## Rôles et Fonctionnalités

### Sportifs (Clients)

- Découvrir les coachs sportifs disponibles sur la plateforme.  
- Consulter les profils des coachs : disciplines, niveau, années d’expérience, certifications.  
- S’inscrire et se connecter.  
- Réserver une séance sportive en choisissant un créneau horaire disponible.  
- Gérer leurs réservations : consultation, modification ou annulation.

### Coach Sportif

- Se connecter pour gérer les demandes de séances.  
- Accepter ou refuser les réservations selon sa disponibilité.  
- Gérer son profil professionnel : photo, biographie, disciplines sportives, certifications, coordonnées.  
- Gérer ses créneaux de disponibilité pour les séances sportives.  
- Afficher des statistiques détaillées :
  - Nombre de demandes en attente  
  - Nombre de séances validées pour la journée  
  - Nombre de séances validées pour le lendemain  
  - Détails du prochain sportif et de sa séance




## Fonctionnalités JavaScript

- **Validation des formulaires** avec Regex : email, téléphone, mot de passe, etc.  
- **Modals dynamiques** : détails des séances, confirmation d’actions, messages d’erreur.  


## Technologies Utilisées

- **Frontend** : HTML, CSS, JavaScript  
- **Backend** : PHP, MySQL
